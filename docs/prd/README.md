# PRD lifecycle

PRDs are organized by lifecycle phase. Every PRD lives in its own folder.

- `Draft/` — initial spec, not yet reviewed.
- `Review/` — under stakeholder review.
- `Approved/` — accepted for implementation.
- `Implemented/` — delivered and verified.
- `Deprecated/` — no longer relevant.

Use `docs/prd/<Phase>/(PRD-NNN)-Title/` for each effort.
