# PRDs (Legacy)

> **Note:** PRDs are a legacy artifact type. New work uses Epics and Agent Specs. See `docs/epic/` and `docs/spec/`.

PRDs are organized by lifecycle phase. Every PRD lives in its own folder.

- `Implemented/` — delivered and verified.
- `Abandoned/` — migrated to Epics/Specs or no longer relevant.

See [list-prds.md](list-prds.md) for the full index.
